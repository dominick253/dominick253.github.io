Dominick Pescetto's Homelab

Hello and welcome to my website!


Here is a brief overview of my homelab. It all started with a broken laptop. I decided to make it into an opnsense router after my old router broke. Since then it has been everything from a kareokee machine to a router, to a home media pc, and now as an ubuntu server running docker containers. Including running this very website, alongside several other services such as jellyfin, portainer, nextcloud, and virtual machines including ubuntu desktop.

Downtime was due to some server "maintanece"

Before I could switch my old laptop into a server I had to make a new router. So I decided to get a pi 4 b and install openWRT on it. I 3d printed the case and added an usb ethernet adapter. So far it has been very good and very reliable. However I could see make a cluster of pi's in the future and using them to run my server for high availability.

Learning every day.
